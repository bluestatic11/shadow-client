What's inside

⚡

Performance stack pre-bundled

Sodium, Lithium, ImmediatelyFast, EntityCulling, ScalableLux, ThreadTweak, FerriteCore, Krypton, Iris, Bobby — installed and tuned for you.

▦

130+ HUD modules

FPS, HP, coords, ping, combat tracking, kill streaks, totem counts, friend lists, server timers — all configurable, all draggable, all themed.

✎

Drag-to-position HUD editor

Open the menu, click ✎ EDIT HUD, drag the card stack anywhere on screen. Scroll wheel resizes. Esc saves.

⚙

Every module is configurable

Click the gear on any card. Pick colors, anchor points, scales, behaviors. Save snapshots, switch profiles.

★

Cosmetics

Textured capes, solid 3D dragon wings, halo, fairies, footstep trails, bow trails — all customizable.

☷

Discord RPC + Shulker tooltips

Show off what you're playing. Hover any shulker box in inventory to see its contents without opening it.
